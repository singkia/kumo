<ComponentSection>
  <Heading level={2}>Usage</Heading>
  <p>
    <code class="rounded bg-kumo-control px-1 py-0.5 text-sm">ColorPicker</code>{" "}
    keeps its external API in hex format. The trigger shows the current color,
    and the popover provides an HSB color area, hue slider, format switching,
    and a hex input that commits on{" "}
    <code class="rounded bg-kumo-control px-1 py-0.5 text-sm">Enter</code> or
    blur.
  </p>

```tsx
import { ColorPicker } from "@cloudflare/kumo";
import { useState } from "react";

export default function Example() {
  const [value, setValue] = useState("#F48120");

  return <ColorPicker value={value} onChange={setValue} />;
}
```

  <p>
    EyeDropper support is progressive enhancement. When the browser exposes the
    API, the panel shows a pick-from-screen action; unsupported browsers still
    keep the rest of the picker working.
  </p>
</ComponentSection>
